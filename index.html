  1. git init: Inicializa git en el repositorio
  2. git add . : Agrega todos los archivos nuevos, modificados despues del commit
  3. git status : Muestra estado de la rama
  4. git branch : Muestra la rama en la que estamos trabajando
  5. git branch -M master : Crea la rama master
  6. git checkout master : Nevegamos entre ramas
  7. git commit -m "Commit 1 "Pagina web inicial": Crear una version (repositorio y rama)
  8. git log: Muestra los commits del repositorio
